What's New · v6.16.4.1

Panic recovery. Idle/sleep control.
Lock screen overhaul.

v6.16.4.1 rolls in 8 versions worth of shipping since v6.16.3.4.2 — lock screen + widget scale + idle cascade + the keybind that replaces force-power-off.

v6.16.4.1 · Hotfix

buildPanic script fixes

v6.16.4 panic had 3 bugs — wiped monitor config (hyprctl reload), killed Quickshell (SIGUSR2), looped music marquee (workspace bounce). All fixed. Single press now preserves Quickshell entirely; only double-press-within-10s restarts it.

v6.16.4 · Laptop

boltSUPER+SHIFT+CTRL+Esc

Escape hatch keybind — fires through frozen hyprlock (Hyprland's bindl flag). Runs 9-step recovery: kill zombie hyprlock, DPMS kick, forcerendererreload, swww revive, input kick, clean relock. ~2 seconds.

v6.16.3.8 · Power

bedtimeIdle & Sleep cascade

Lock after idle (30s / 1m / 30m / 1h / 3h / 5h / Never). Sleep after idle (same options). Lid close action (Sleep / Lock only / Do nothing). zen-hypridle-sync.sh rewrites hypridle.conf live.

v6.16.3.7 · Widgets

aspect_ratioWidget Scale slider

Settings → Widgets → Widget Scale. Drag from 0.5× to 2.0× and all three desktop widgets (clocks, weather, sysmon) resize in lockstep. 68 scale multipliers baked in via single _scale property.

v6.16.3.6 · Lock

lockLock screen overhaul

Clock font now matches your bar widget font (Black/Bold weight per family). No "Heyah Username" greeting. Weather mood line + rotating care message, gender-aware (Neutral / Male / Female in Personal Preferences).

v6.16.3.6 · Popups

infoHover popup parity

ZenClock + ZenSysMonitor now use proper PopupWindow with 350ms hover-intent. Clock shows live time + ISO week + timezone. SysMonitor shows CPU/GPU/Memory/Network sections. Identical styling across both.

v6.16.3.5 · Start

imageStart Menu logo picker

7 bundled SVG logos (Arch / CachyOS / EndeavourOS / Fedora / Ubuntu / NixOS / generic Linux). Auto-detect from /etc/os-release, or pick manually. Custom image upload supported. 4-column grid picker.

v6.16.3.4.4 · Input

brightness_6ROG brightness detection

Laptop backlight detection extended — amdgpu_bl0 and asus::kbd_backlight now detected. Themes dropdown made scrollable (was cutting off Paper / Black / Navy). Part of the v6.16.3.4.x hotfix sub-series.

v6.16.3.4.3 · Settings

animationAnimations live-reload + scroll

Animations page now reflects current hyprctl state (was showing stale defaults). Scroll-to-bottom fix — last rows no longer cut off by ScrollView. Input toggle design synced with System Tray toggle design.

v6.16.3.4.1 · Widgets

saveWidget color persistence

Weather + sysmon widget background colors (mode, custom color, opacity) now persist across shell restart. Previously reverted to defaults on every restart — frustrating if you'd just picked your preferred theme.

Quick Start

Up and running in minutes.

Bootstrap mode is safe for KDE, GNOME, and COSMIC users.

bash · main branch · v6.16.4.1 stable 
# Clone the repo
git clone https://github.com/Gekinzen/zen_barebone_alpha_development.git
cd zen_barebone_alpha_development

# Pin to the exact v6.16.4.1 tag — or stay on main
git fetch --tags
git checkout v6.16.4.1     # pinned
# — or —
git checkout main          # always latest stable

# --bootstrap: installs Hyprland + all deps, safe for KDE/GNOME/COSMIC
chmod +x install.sh
./install.sh --bootstrap

# Already on Hyprland? Skip --bootstrap:
./install.sh

# Verify panic keybind is wired
hyprctl binds | grep -i zen-panic

# Test it (safe — no damage)
~/.local/bin/zen-panic.sh
cat ~/.cache/zen-shell/panic.log | tail -30
