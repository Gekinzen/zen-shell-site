金継ぎ Kintsugi v6.16.4.11.2 · stable

A QML-native desktop environment for Hyprland.

Zen Shell replaces GTK4, Python, and Waybar with a unified Quickshell architecture. Panel, control center, wallpaper engine, themes, widgets — all one process, one language, one design system. Built for Arch and CachyOS. Extensively tested on AMD Ryzen + Radeon. Runs on Hyprland 0.54+.

Kintsugi (金継ぎ) — the Japanese art of repairing broken pottery with gold. The seams aren't hidden; they're illuminated. This release took eleven iterations to land — the color picker alone needed four attempts. What you get is the bowl, lacquered in gold.

Project archive

From sprout to lacquered bowl.

Zen Shell began as Zen Barebone Alpha — a bare Waybar + Python concept on Hyprland 0.52. What you use today is the third full architecture. Every era preserved here for reference.

Alpha v0.91 Genesis · Wakaba (若葉)

Wakaba — the first sprout.

The earliest concept build. Bare Waybar configuration, a handful of Python helpers, rofi for launching, Hyprland running the show. No unified theme system yet — each component styled independently. Proof-of-concept that became the seed of everything after.

Hyprland 0.52 · Waybar · Python + rofi
Alpha v2.x (v2.1.3) Legacy · Koke (苔)

Koke — moss grows steady.

The full Python / GTK4 / Libadwaita era. "Zen Barebone Alpha" — a complete desktop ecosystem with a custom GTK control center (Hypr Control Center), custom dock module, unified theme engine covering Waybar / Rofi / Kitty / SwayNC / GTK4 widgets, desktop widgets (clock, weather, system monitor), smart start menu, welcome screen on first boot. Everything was Python + GTK4, talking to Hyprland via hyprctl. This was the bowl that existed before Kintsugi broke and re-lacquered it.

v6.10 – v6.14 Rewrite · Yugen (幽玄)

Yugen — subtle, profound grace.

The QML rewrite cycle. GTK4 gradually replaced with Quickshell-native QML. v6.10 shipped the new foundations. v6.14 added theme switching, panel modes, control center. This era marked the architectural shift that unlocked everything after — the foundational grace beneath all subsequent polish.

Hyprland 0.53+ · Quickshell QML · JSON bridge to GTK
v6.15.x series Unified · Ensō (円相)

Ensō — the circle closes.

v6.15.1 through v6.15.15 — full Quickshell-native stack. Bar, Start Menu, Control Panel, Settings UI with HMSection/HMRow components, Theme engine, Wallpaper manager, audio-reactive music strings (cava + playerctl), screenshot rope overlay, avatar system, island mode, system tray. Everything spoke one language for the first time. The scattered modules closed into one circle.

Hyprland 0.53+ · 15-drop evolution · Music strings · Screenshot ropes
v6.16.x through v6.16.3.x Refinement · Ma · Shibui · Sabi

The architecture matures.

v6.16 Ensō base — battery, power profiles, volume OSD, GPU switcher. v6.16.1 Ma (間) — cascade Control Panel, two-column layouts. v6.16.2.3 Shibui (渋い) — click-through masks, OpacityMask avatar, default wallpaper, mouse sensitivity, bulletproof single-instance installer. v6.16.3 Sabi (寂) — Material power icons, lid-close patch, PowerBadge, Start Menu logo picker, Lock screen overhaul (font sync + weather mood + gender-aware care messages), Widget Scale slider, Idle/Sleep/Lid unified UX.

Hyprland 0.54+ · 3-subseries refinement · 30+ tagged releases
v6.16.4.x · v6.16.4.11.2 Current · Kintsugi (金継ぎ)

Kintsugi — gold in the seams.

The current release. v6.16.4 shipped the Panic Recovery keybind (SUPER+SHIFT+CTRL+Esc — works through frozen hyprlock) and hardened resume pipeline. v6.16.4.1 hotfixed the panic script. The 4.x cycle then ran for 11 alpha iterations over 2 days — widget scale awareness, gaps preservation, Start Menu breathing, terminal auto-detect, Dark Mode toggle, WiFi Connect rewrite, and four attempts at the color picker before Quickshell's PopupWindow primitive finally stuck. v6.16.4.11.2 — palette relocation, Material-style dropdown with WCAG luminance contrast, PaletteBox component. This is the bowl lacquered in gold.
