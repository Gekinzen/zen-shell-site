What's New · v6.16.2.3.x

Click-through transparency
and a bulletproof installer.

Closeout of the v6.16.2.3 hotfix series — every fix from .3.1 → .3.6 rolled in.

v6.16.2.3.6 · Installer

terminalSingle-Instance Launch

Bulletproof kill loop — SIGTERM ×3 then SIGKILL ×2, plus survivor verification. If anything survives, install.sh refuses to spawn another. No more stacked duplicate bars on re-install.

v6.16.2.3.6 · Avatar

account_circleOpacityMask Circle

The original GLSL shader silently failed on some Qt builds. Replaced with OpacityMask from Qt5Compat.GraphicalEffects — same module ZenStrings already uses. Versioned filenames + pruning + diagnostic log to /tmp/zen-avatar-debug.log.

v6.16.2.3.2 · UX

touch_appSettings + CP Click-Through

Both windows use mask: Region { item: zenSettingsPanel } so the transparent backdrop passes clicks through to apps below. Close via ✕ or Super+C — matches macOS / GNOME behavior.

v6.16.2.3.1 · Bar

graphic_eqMusic Rope Click-Through

mask: Region {} on stringsWindow makes the audio-reactive bar overlay fully input-transparent. Browsers, editors, and file managers below the strings now receive clicks normally.

v6.16.2.3.1 · Clock

scheduleHover Peek + Scroll Wheel

Peek tooltip after 350ms hover. WheelHandler cycles calendar months. Right-click cycles through clock formats. Same wheel handler attached to the open calendar itself.

v6.16.2.3.1 · Persistence

saveIsland Mode Survives Reboot

panelStateLoaded() signal gates nuclear restart against the startup-load transient. Initial-load mode transitions no longer trigger a restart cascade — Island mode actually persists now.

v6.16.2.3.2 · Wallpaper

cloud_downloadDefault + Repo Browser

Fresh installs auto-fetch the default wallpaper from Gekinzen/images-demo via swww. New WallpaperRepoService.qml singleton renders an "Online" tab in the picker — one-click download + apply, with offline-capable cache.

v6.16.2.3.2 · Input

mouseMouse Sensitivity Controls

New MouseSettingsService singleton — sensitivity, scroll factor, two natural-scroll toggles. Live via hyprctl keyword, persisted to ~/.config/hypr/zen-mouse.conf. Settings page + Control Panel tab both bound to the same singleton.

Roadmap · v6.16.x → v6.16.3

Phases shipped. Next stop, v6.16.3.

v6.16.0 → v6.16.2 are all shipped. The current beta is beta-v12.6.16.2.3.6 — the closeout of the v6.16.2.3 hotfix series. Once it stabilizes the prefix is dropped and the branch promotes to v6.x.x.x on main. v6.16.3 is the next feature phase.

Quick Start

Up and running in minutes.

Bootstrap mode is safe for KDE, GNOME, and COSMIC users.

bash · main branch · official release 
# Clone the repo
git clone https://github.com/Gekinzen/zen_barebone_alpha_development.git
cd zen_barebone_alpha_development

# Already on main (stable v6.15.13) by default
# --bootstrap: installs Hyprland + all deps, safe for KDE/GNOME/COSMIC
chmod +x install.sh
./install.sh --bootstrap

# Already on Hyprland? Skip --bootstrap:
./install.sh
bash · beta-v12.6.16.2.3.6 · recommended for laptop users 
# Clone the repo
git clone https://github.com/Gekinzen/zen_barebone_alpha_development.git
cd zen_barebone_alpha_development

# Switch to the latest beta — v6.16.2.3.6
# Includes click-through Settings/CP, OpacityMask avatar,
#           default wallpaper, mouse sensitivity, single-instance install
git checkout beta-v12.6.16.2.3.6

# --bootstrap for fresh installs (safe alongside KDE/GNOME/COSMIC)
chmod +x install.sh
./install.sh --bootstrap

# Already on Hyprland? Skip --bootstrap:
./install.sh

# Verify single instance after install
pgrep -fa 'quickshell.*zen-shell' | wc -l   # should print 1
