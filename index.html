v6.16.4.11.2 · Current Stable

A QML-native desktop environment for Hyprland.

Zen Shell replaces GTK4, Python, and Waybar with a unified Quickshell architecture. Panel, control center, wallpaper engine, themes, widgets — all one process, one language, one design system. Built for Arch and CachyOS. Extensively tested on AMD Ryzen + Radeon. Runs on Hyprland 0.54+.

Showcase

Zen Shell, in practice.

The current UI as shipped in v6.16.4.11.2. Panel, Start Menu, Settings, Control Panel, Wallpaper browser, Theme engine. Click any image to open full-resolution.

Panel
Status bar Configurable modules · island mode · system tray Workspace indicators, clock with hover peek and scroll-wheel month cycle, system tray with hover tooltips. Bar modules configurable from Settings → Bar Modules.
Super+A
Start Menu Pinned tiles · app grid · sys-info footer Pinned tiles (72px, breathing room via v4.5). Live search. Sys-info popover with Hyprland version / kernel / DMI device / BIOS from /sys/class/dmi/id.
New · v4.11.2
Themes page PaletteBox · Custom Themes manager · WCAG dropdown Theme palette relocated from General. Click any 60×60 box to open PopupWindow picker with live hex typing. Save / Rename / Delete custom themes below.
Super+C
Control Panel Connectivity · Power · Audio · Dark Mode · Input Quick toggles for WiFi / Bluetooth / Audio + Power Profile + Gaming Boost + Dark Mode (syncs GTK3/4 + libadwaita + gsettings). Tap-through backgrounds via mask:Region.
New · v4.10
Color Picker PopupWindow · HS canvas · L slider · live hex Fourth iteration. Uses Quickshell PopupWindow primitive backed by xdg_popup — compositor handles positioning, zero coord math. Hex textbox commits live on every valid keystroke.
Super+W
Wallpaper Picker Local grid · Online repo · change interval 4-column grid (v4.6, was 5). "Online" toggle browses Gekinzen/images-demo via GitHub API. Change interval, random transition, blue border = current.
