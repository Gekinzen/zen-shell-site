A QML-native desktop environment for Hyprland.

Zen Shell replaces GTK4, Python, and Waybar with a unified Quickshell architecture. Panel, control center, wallpaper engine, themes, widgets — all one process, one language, one design system. Built for Arch and CachyOS. Extensively tested on AMD Ryzen + Radeon. Runs on Hyprland 0.54+.

Modori (戻り) — "to return". After the Tategaki vertical-bar attempt hit three startup-blocking parser errors and a broken empty-bar render, the bar code rolled back to the proven Tachiagari .7.1 base. Modori is what was added on top: smart-contrast theme engine (WCAG-aware), in-shell WiFi password prompt at WlrLayer.Overlay, GTK Dark Mode toggle, redesigned WiFi+BT panels, and two new built-in themes with paired enso ink-wash wallpapers. Promoted to stable after the .11 and .12 reliability patches landed.

v6.16.4.12.9.10Current Stable · Modori (戻り)

Modori — to return.

The current stable. After the Tategaki vertical-bar attempt hit three startup-blocking parser errors and a broken empty-bar render, the bar code rolled back to the proven Tachiagari .7.1 base. Modori is what was added back on top: smart-contrast theme engine (WCAG-aware, every loaded theme runs through a luminance check), in-shell WiFi password prompt at WlrLayer.Overlay (replaces zenity that was hiding behind the Control Panel), redesigned WiFi+BT panels (saved/available split, larger 44px tap targets, scan toggle, pair flow), GTK Dark Mode toggle (one-tap atomic sync of gsettings + GTK3/4 + libadwaita), bulletproof sidebar user labels (env-fallback resolution), and two new built-in themes — Modori Dark and Modori Light — with paired procedural enso ink-wash wallpapers. The .11 patch added WiFi route-metric preference (wifi wins over LAN on user-tap), preventStealing on row taps, and the open-network parser fix. The .12 patch was a critical one-line restore of a recursive helper that was silently breaking every WiFi/BT/audio toggle.
