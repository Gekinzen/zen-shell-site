What's New · v6.16.x

Laptop-grade power management
and multi-GPU support

The biggest feature release since Music Strings in v6.15.

v6.16 · Battery module

battery_charging_fullBattery Module

Bar + SysRow integration. Three display modes — icon, text, progress bar. Auto-hides on desktops. Color shifts at 10/30/50% thresholds.

v6.16 · Power profiles

electric_boltPower Profile Service

Singleton wrapper around powerprofilesctl. Saver / Balanced / Performance switchable from Control Panel. Persists across reboots.

v6.16 · GPU Switcher

memoryGPU Switcher Service

4 modes: Auto, Force Integrated, Force Dedicated, Auto+Gaming. Auto-detects NVIDIA/AMD/Intel. Writes env vars for systemd.

v6.16 · Gaming

sports_esportsGaming Boost

One-tap toggle forces performance profile + disables compositor blur/dim/animations. Auto-gaming daemon watches for Steam, Lutris, Wine, Proton.

v6.16.1 · UI

view_sidebarCascade Control Panel

Auto-splits into two columns when expanded content exceeds viewport height. Click-outside-to-close. Matches macOS Control Center UX.

v6.16.1 · Design

toggle_onUnified HMSwitch

New centralized pill toggle component. All 27 toggles across the shell now render identically with 150ms OutCubic animation and theme-reactive colors.

v6.16.1 · Widgets

drag_panSmooth Widget Drag

Root cause found after 5 iterations — _applyPositions() was resetting position mid-drag. Fixed with _anyDragActive guard. No more ghost trails.

v6.16.1 · Widgets

color_lensWidget Background Picker

Per-widget background: Default / Theme-synced / Custom. 10 preset swatches + opacity slider for Weather and System Monitor widgets.

Architecture

Modular QML structure

73+ files. Clean separation of services, UI, and state.

├── shell.qml                  ── entry point · bar · overlays · widgets
├── Bar.qml                    ── bottom bar with module loader
├── Battery.qml                ── battery bar module            ← v6.16
├── ZenStrings.qml             ── audio-reactive visualizer      ← v6.15
├── ZenScreenshotOverlay.qml  ── region screenshot + ropes      ← v6.15
├── PowerProfileService.qml   ── powerprofilesctl + gaming boost ← v6.16
├── GPUSwitcherService.qml    ── GPU selection + env vars       ← v6.16
├── HMSwitch.qml              ── unified pill toggle (×27)      ← v6.16.1
├── ThemeService.qml
├── SettingsStateV2.qml        ── full hyprland state persistence
├── ControlPanel.qml
├── ZenSettings.qml            ── settings window · 14 pages
├── BatterySettingsPage.qml   ── battery · power · GPU page     ← v6.16
└── ...                        ── ~73 QML files total

~/.local/bin/
├── zen-screenshot.sh          ── screenshot pipeline
├── zen-volume-notify.sh       ── volume/brightness OSD
├── zen-game-watcher.sh       ── auto-gaming process detection  ← v6.16
└── prime-run                 ── one-shot dGPU launcher         ← v6.16
Quick Start

Up and running in minutes

Bootstrap mode is safe for KDE, GNOME, and COSMIC users.

bash · main branch · official release 
# Clone the repo
git clone https://github.com/Gekinzen/zen_barebone_alpha_development.git
cd zen_barebone_alpha_development

# Already on main (stable v6.15.13) by default
# --bootstrap: installs Hyprland + all deps, safe for KDE/GNOME/COSMIC
chmod +x install.sh
./install.sh --bootstrap

# Already on Hyprland? Skip --bootstrap:
./install.sh
bash · beta-v12.6.16.1.11 · recommended for laptop users 
# Clone the repo
git clone https://github.com/Gekinzen/zen_barebone_alpha_development.git
cd zen_barebone_alpha_development

# Switch to the beta branch (v6.16.1.11)
# Includes: Battery, Power Profiles, GPU Switcher, Gaming Boost,
#           Multi-GPU tabs, Cascade Control Panel, HMSwitch, OSD
git checkout beta-v12.6.16.1.11

# --bootstrap for fresh installs (safe alongside KDE/GNOME/COSMIC)
chmod +x install.sh
./install.sh --bootstrap

# Already on Hyprland? Skip --bootstrap:
./install.sh
